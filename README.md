# Moonstripe

Moonstripe is a classic Winstripe-inspired complete theme for Pale Moon.

It is continued from the abandoned Winstripe Moon theme and updated for current Pale Moon versions.

## Features

* classic Winstripe-style browser appearance
* Pale Moon complete theme
* updated Permissions Manager styling
* updated Add-ons Manager category icons
* updated Page Info icons
* updated Status Bar Preferences icons
* scrollbar, URL bar, search bar and downloads UI fixes
* cleanup and release packaging by Halvar666

## Compatibility

Moonstripe targets Pale Moon:

* minimum version: 27.0.0b3
* maximum version: 34.*

## Installation

Download the `.xpi` file and install it in Pale Moon as a complete theme.

After installation, open:

`Tools` → `Add-ons` → `Appearance`

Then select **Moonstripe** and restart Pale Moon if needed.

## Build

From the repository root:

```bash
./build.sh
```

The generated XPI will be created in the `dist/` directory.
