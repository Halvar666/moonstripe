# Credits

Moonstripe is a continuation/fork of the abandoned Winstripe Moon theme for Pale Moon.

## Current fork

### Halvar666
Moonstripe continuation, Pale Moon compatibility adaptation, release cleanup, and new development work.

Work in this fork includes:

- rename/release preparation from Winstripe Moon to Moonstripe
- Permissions Manager geometry and icon integration
- Add-ons Manager category icon updates
- Page Info icon sprite replacement/redraw
- Status Bar Preferences icon work
- scrollbar tuning
- URL bar, search bar, autocomplete, downloads popup and favicon containment fixes
- missing asset backfill
- release cleanup and packaging
